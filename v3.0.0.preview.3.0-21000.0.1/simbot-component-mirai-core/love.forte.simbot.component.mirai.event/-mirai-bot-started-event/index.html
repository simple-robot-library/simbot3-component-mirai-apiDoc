当一个 MiraiBot 执行 MiraiBot.start 时会触发此事件。

MiraiBot.start 会在最后完成整个事件触发流程后返回，因此对此事件的处理请尽可能避免过长时间的阻塞或挂起， 以及尽量避免出现嵌套执行 start.
